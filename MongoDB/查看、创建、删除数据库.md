### 命令

MongoDB 创建数据库的语法格式如下：

```
use DATABASE_NAME
```

```
> show dbs
admin   0.000GB
config  0.000GB
local   0.000GB
> use test
switched to db test
> show dbs
admin   0.000GB
config  0.000GB
local   0.000GB
> use houjichao
switched to db houjichao
> db
houjichao
> db.houjichao.insert({"houjichao":"test"})
WriteResult({ "nInserted" : 1 })
>
> show dbs
admin      0.000GB
config     0.000GB
houjichao  0.000GB
local      0.000GB
```

MongoDB 中默认的数据库为 test，如果你没有创建新的数据库，集合将存放在 test 数据库中。

> **注意:** 在 MongoDB 中，集合只有在内容插入后才会创建! 就是说，创建集合(数据表)后要再插入一个文档(记录)，集合才会真正创建。
