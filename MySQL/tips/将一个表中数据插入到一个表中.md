### 将一个表中数据插入到另一个表中

```
REPLACE INTO index_his_value_latest ( object_id, index_id, VALUE, create_at ) SELECT
object_id,
index_id,
substring_index( group_concat( `value` ORDER BY create_at DESC ), ",", 1 ) `value`,
max( create_at ) create_at 
FROM
	index_his_value 
GROUP BY
	object_id,
	index_id;
```
